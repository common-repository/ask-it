== Description ==

Ask It enables wordpress users to ask administrators questions and have them answered via the dashboard. Three dashboard widgets are added:

1. Question Form
1. My Questions
1. Answered Questions

Other features include:

* Options to remove any or all of the default dashboard widgets
* Email & Text alerts when questions are asked, and email alerts when questions are answered

== Installation ==

1. Upload `/ask-it/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
